# Código de conducta

Este repositorio busca ser un espacio colaborativo, respetuoso y orientado al aprendizaje.

## Comportamiento esperado

- Tratar a otras personas con respeto.
- Dar feedback técnico de forma clara y constructiva.
- Aceptar correcciones cuando mejoren el material.
- Mantener discusiones enfocadas en el contenido del repositorio.
- Evitar comentarios despectivos, ataques personales o burlas.

## Comportamiento no aceptado

- Acoso, insultos o discriminación.
- Publicar contenido ofensivo o ajeno al objetivo del repositorio.
- Descalificar contribuciones sin explicación técnica.
- Compartir soluciones falsas intencionalmente.

## Reportes

Si ves una conducta inapropiada, abre un issue o contacta a quienes mantienen el repositorio. Los reportes se revisarán con criterio y buscando proteger el ambiente de colaboración.
