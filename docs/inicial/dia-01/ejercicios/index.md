# Ejercicios — Día 1

## 0 — Hello World

**Link:** Link Codeforces del problema

### Idea (sin código)

El problema pide imprimir un mensaje fijo. No hay decisiones que tomar ni datos que procesar: la respuesta siempre es la misma. Este tipo de ejercicio sirve para comprobar que el entorno compila, ejecuta y muestra la salida correctamente.

### Conceptos clave a revisar

- **Algoritmo ad hoc:** solución construida directamente para este caso, sin depender de una técnica clásica estandarizada.
- **Salida estándar:** mecanismo usado para mostrar la respuesta del programa en consola o en el juez online.
- **Flujo secuencial:** ejecución paso a paso sin ciclos ni decisiones condicionales.

### Conceptualización técnica

La solución consiste en escribir directamente la cadena solicitada en la salida estándar.

1. Preparar la salida del programa.
2. Imprimir `Hello World!`.
3. Terminar la ejecución.

**Complejidad:** `O(1)` tiempo y `O(1)` memoria.
