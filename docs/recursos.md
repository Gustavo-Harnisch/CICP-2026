# Recursos

Enlaces y referencias generales para complementar las clases y practicar
entre jornadas. Ajusta o quita lo que no aplique a tu campamento.

## Jueces y plataformas

- [Codeforces](https://codeforces.com/)
- [Kattis](https://open.kattis.com/)
- [AtCoder](https://atcoder.jp/)
- [SPOJ](https://www.spoj.com/)

## Editores y compiladores online

- [Compiler Explorer](https://godbolt.org/) — ver el ensamblador generado.
- [Wandbox](https://wandbox.org/) — compilar C++ rápido sin instalar nada.

## Referencia y teoría

- [cp-algorithms.com](https://cp-algorithms.com/) — algoritmos explicados con implementación en C++.
- [USACO Guide](https://usaco.guide/) — guía progresiva por nivel.
- [Competitive Programmer's Handbook](https://cses.fi/book/book.pdf) — libro gratuito de Antti Laaksonen.

## Plantilla de código

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);

    // Código aquí

    return 0;
}
```

!!! tip "Antes del contest"
    Verifica que tu plantilla, el `.vimrc` o snippets del editor y los
    atajos de tu IDE estén listos antes de que empiece cada bloque de
    contest, para no perder tiempo configurando.
